# KOC AI增长助手 - 腾讯PCG校园AI产品创意大赛参赛作品

## 📋 项目概述

**参赛赛道**：赛道⑤ - PCG技术线-智能营销/AI小游戏方向  
**产品名称**：KOC AI增长助手  
**产品定位**：面向普通KOC的社媒AI Agent，通过AI能力提供内容方向、选题建议、发布策略、互动优化等服务  

---

## 🎯 产品背景

### 用户痛点
如今大量普通KOC在视频号等社交媒体平台存在：
1. **内容定位模糊** - 不知道自己适合做什么方向
2. **选题低效** - 花费大量时间思考拍什么
3. **运营不专业** - 不懂平台算法和发布策略
4. **难以快速涨粉** - 缺乏系统性的成长路径

### 解决方案
KOC AI增长助手通过集成大语言模型（支持所有OpenAI兼容API），为KOC提供：
- 🤖 **AI账号诊断** - 深度分析账号定位与内容策略
- 💡 **智能选题引擎** - 基于热点趋势与账号定位的个性化选题推荐
- ✍️ **内容创作助手** - 一键生成视频脚本、文案、标题等
- 📊 **数据看板** - 追踪账号成长，AI提供优化建议

---

## ✨ 核心功能

### 1. API配置中心
- 支持所有OpenAI兼容API（OpenAI、混元、DeepSeek、Claude等）
- 用户可自定义API端点、密钥、模型名称和温度参数
- 提供预设配置快速切换
- 配置仅保存在本地浏览器，保护用户隐私

### 2. AI账号诊断
**功能描述**：用户输入账号名称和简介后，AI深度分析：
- 账号定位分析与改进建议
- 内容策略规划（核心/互动/轻松内容配比）
- 受众画像推测
- 30天涨粉计划
- 账号健康度综合评分（可视化展示）

### 3. 智能选题引擎
**功能描述**：基于内容赛道和创意程度，AI生成选题建议：
- 支持8大内容赛道：生活方式、科技数码、美食探店、旅行户外、时尚美妆、健身运动、知识教育、娱乐八卦
- 三个创意程度：稳健实用、平衡创新、大胆创新
- 展示选题难度等级
- 结合今日热点标签

### 4. 内容创作助手
**功能描述**：提供四大创作工具：

#### 4.1 视频脚本生成
- 输入视频主题、时长、风格倾向
- AI生成完整分镜脚本（开头吸睛、核心价值、互动引导、结尾升华）
- 支持专业严谨/幽默轻松/情感共鸣/直截了当四种风格

#### 4.2 标题优化
- 基于内容主题生成多个吸引人的标题
- 提供吸引力评分
- 支持多平台适配（视频号、抖音、小红书、B站）

#### 4.3 文案生成（Demo中）
#### 4.4 封面建议（Demo中）

### 5. 数据看板
**功能描述**：可视化展示账号成长数据
- 核心指标卡片：粉丝数、播放量、点赞数、评论数（含周增长）
- 粉丝增长趋势图（需集成Chart.js）
- 内容表现分析图（需集成Chart.js）
- AI数据洞察与建议
- 内容发布日历

## 🛠️ 技术架构

### 前端技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式设计（渐变色、模糊效果、响应式布局）
- **Vanilla JavaScript** - 交互逻辑
- **Font Awesome** - 图标库

### AI集成方案
- **API兼容性**：支持所有OpenAI API格式的接口
- **灵活性**：用户可自定义endpoint、api_key、model、temperature
- **扩展性**：可接入混元、DeepSeek、Claude等多种大模型

### 数据存储
- **LocalStorage** - 保存用户API配置
- **无后端设计** - Demo版本纯前端实现，便于快速部署

## 🚀 部署与使用

### 本地运行
1. 进入项目目录
2. 启动HTTP服务器（任选一种）：
   ```bash
   # Python
   python -m http.server 8080
   
   # Node.js
   npx http-server -p 8080
   ```
3. 浏览器访问 `http://localhost:8080`

### 线上部署
可部署至：
- **GitHub Pages**（免费静态托管）
- **Vercel**（推荐，支持自定义域名）
- **腾讯云静态网站托管**
- **Netlify**

## 💡 创新点

### 1. 开放API架构
不同于封闭的商业产品，本产品支持用户使用自己的API Key，可接入任意OpenAI兼容的大模型，包括：
- OpenAI GPT-4/3.5
- 腾讯混元
- DeepSeek
- Claude（通过代理）
- 其他自建模型

### 2. 纯前端实现
- 无需后端服务器，降低部署成本
- 用户API密钥仅保存在本地，提升隐私安全
- 可静态托管，访问速度快

### 3. 数据驱动的内容策略
不仅提供内容创作工具，更通过数据分析提供：
- 最佳发布时间建议
- 内容类型效果对比
- 受众画像分析
- 涨粉路径规划

### 4. 与PCG业务结合
- 可集成视频号API，直接发布内容
- 可接入腾讯广告智能投放平台
- 可结合AIGC能力自动生成广告素材
