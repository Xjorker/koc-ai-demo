**画面：架构图（提前做好PPT/图示）**

**（口播）**

> 最后介绍一下技术架构。
>
> 本项目采用**前后端分离**架构：
> - **前端**：纯 HTML5 + CSS3 + JavaScript，单页应用，无需任何框架
> - **后端**：Python FastAPI，提供 RESTful API
> - **Agent框架**：使用 LangGraph 实现状态机驱动的 Agent 编排
> - **AI层**：通过统一的 AI Client 对接 OpenAI 兼容 API，支持灵活切换不同模型

**架构图文字**：

```

│   :8080     │    JSON       │   :8000     │    JSON         │             │
```

## 【录制注意事项】

### 录制前准备
1. ✅ 提前配置好 API，确保 AI 调用能正常工作
2. ✅ 准备好转场文字/图片（架构图、创新点）
3. ✅ 确保网络稳定，AI 响应时间适中
4. ✅ 测试录音环境，避免噪音

### 录制工具推荐
- **剪映**（推荐）：支持屏幕录制、剪辑、BGM添加，一站式搞定
- **OBS**：专业录屏软件
- **PowerPoint录制**：直接录制演示模式

### 剪辑建议
- 添加转场动画（推拉、淡入淡出）
- 背景音乐音量调低，不盖过人声
- 语速适中，每分钟180-200字左右
- 关键操作可以放慢或暂停一下

### 备用方案
- 如果 AI 响应太慢，可以提前录好结果，演示时直接展示
- 重要功能多录几遍，选最好的一条
