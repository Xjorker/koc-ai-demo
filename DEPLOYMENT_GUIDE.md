## 🚀 线上部署（必须完成）

### 方案A：部署到Vercel（推荐）

#### 步骤1：安装Vercel CLI
```bash
npm install -g vercel
```

#### 步骤2：登录Vercel
```bash
vercel login
```

#### 步骤3：部署项目
```bash
cd c:\Users\王梓亦\WorkBuddy\20260505210514\koc-ai-demo
vercel --prod
```

#### 步骤4：获取线上链接
部署成功后，Vercel会提供一个链接，例如：
```
https://koc-ai-assistant.vercel.app
```

## 🎬 产品录屏制作指南

### 录制要求
- **时长**：3分钟以内
- **格式**：MP4
- **比例**：16:9 横屏
- **内容**：产品介绍 + 功能演示

### 推荐录制工具
1. **OBS Studio**（免费，专业）
   - 下载：https://obsproject.com/
   - 设置：输出分辨率 1920x1080，帧率 30fps

2. **Windows 10/11 游戏栏**（内置）
   - 快捷键：`Win + G`
   - 点击录制按钮

3. **腾讯会议/企业微信**（简便）
   - 开启云录制
   - 后期下载MP4文件

### 录屏脚本建议（3分钟）

#### 0:00 - 0:30 产品介绍
- 展示产品首页
- 说明目标用户和价值主张
- 强调支持自定义OpenAI兼容API

#### 0:30 - 1:30 核心功能演示
1. **API配置**（10秒）
   - 展示配置界面
   - 选择预设配置（混元/DeepSeek等）

2. **账号诊断**（30秒）
   - 输入账号信息
   - 展示AI生成的诊断报告
   - 强调可视化评分

3. **智能选题**（20秒）
   - 选择内容赛道
   - 生成选题列表
   - 展示热点标签

4. **内容创作**（20秒）
   - 生成视频脚本
   - 展示标题优化功能

#### 1:30 - 2:30 技术架构说明
- 展示代码结构
- 说明OpenAI兼容性
- 强调纯前端实现的优势

#### 2:30 - 3:00 创新点与总结
- 开放API架构
- 与PCG业务结合点
- 未来规划

## 📄 说明文档PDF制作

### 方法1：使用浏览器打印（最简单）

1. 打开 `README.md` 的预览（或使用Markdown编辑器）
2. 浏览器按 `Ctrl + P`
3. 选择"另存为PDF"
4. 调整页面设置为A4
5. 保存为 `说明文档.pdf`

### 方法2：使用在线转换工具

1. 访问 https://www.markdowntopdf.com/
2. 上传 `README.md`
3. 下载生成的PDF

### 方法3：使用Word转换（最精美）

1. 将 `README.md` 导入 Microsoft Word
2. 调整格式和样式
3. 导出为PDF
